# Uso

## Instalação Local

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -e ".[dev]"
```

## Configurar Ambiente

Copie `.env.example` para `.env` e configure:

```txt
GITHUB_TOKEN=ghp_replace_with_a_readonly_token
ALLOWED_REPOSITORIES=Cosmess/mcp-github-pr-reviewer
```

## Rodar Testes

```bash
python -m pytest
python -m ruff check .
```

## Rodar O MCP Server

```bash
mcp-github-pr-reviewer
```

## Rodar Com Docker Compose

```bash
docker compose build
docker compose run --rm mcp-github-pr-reviewer
```

## Habilitar Análise LLM Opcional

O analisador padrão é determinístico e local. Para adicionar uma seção de revisão
com um provedor OpenAI-compatible:

```txt
LLM_ANALYZER_ENABLED=true
LLM_API_KEY=replace_with_provider_key
LLM_API_BASE_URL=https://api.openai.com/v1
LLM_MODEL=gpt-4.1-mini
```

Depois chame:

```txt
generate_markdown_review
```

O Markdown gerado incluirá uma seção extra chamada `Análise LLM opcional`.

## Comentário Com Dry-Run

Use `comment_on_pull_request` primeiro em modo seguro:

```json
{
  "owner": "Cosmess",
  "repo": "mcp-github-pr-reviewer",
  "pull_number": 1,
  "body": "## Resumo técnico\n\nReview generated by MCP.",
  "dry_run": true,
  "confirm": false
}
```

Para publicar um comentário real no GitHub, configure:

```txt
ENABLE_GITHUB_WRITE_ACTIONS=true
```

E chame a tool com:

```json
{
  "dry_run": false,
  "confirm": true
}
```
